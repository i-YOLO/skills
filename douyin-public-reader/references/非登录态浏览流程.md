# 非登录态浏览流程

## 目标

确保读取抖音页面时不使用已有登录态，只读取公开可见信息。无法确认非登录态时，必须停止或降级说明，不得假装严格无痕。

## 优先访问方式

1. 优先使用真正的新无痕、私密或隔离浏览器上下文。
2. 如果工具无法创建真正隔离上下文，只能使用未登录公开视角，并在结果中标注：`当前环境无法确认严格无痕，仅按未登录公开视角执行。`
3. 如果当前浏览器已经登录抖音，不得继续使用该页面读取数据。
4. 如果无法判断是否登录，要求用户在非登录或无痕窗口中打开目标页面，或只输出“无法确认非登录态”。

## 非登录态可接受信号

满足以下信号时，可以按未登录公开视角继续：

- 页面顶部显示“登录”按钮。
- 未出现当前用户头像、个人账号菜单、个人关注流、私信未读等账号态信息。
- 页面内容来自公开主页、公开视频详情页或公开搜索结果。

出现以下信号时，必须停止使用该页面：

- 页面显示当前用户头像或账号菜单。
- 页面出现个人关注流、私信、通知、投稿后台等账号态区域。
- 页面能够访问仅登录用户可见的信息。
- 页面要求扫码登录、验证码登录或密码登录后才能继续。

## 登录弹窗处理

遇到登录弹窗时：

1. 只允许尝试关闭弹窗。
2. 关闭后重新检查页面是否仍为未登录公开视角。
3. 如果无法关闭弹窗，停止读取并报告“登录弹窗阻断”。
4. 不得扫码登录、输入手机号、输入验证码或输入密码。

## 主页读取流程

1. 打开用户提供的抖音主页链接。
2. 完成非登录态检查。
3. 只读取主页公开可见区域。
4. 只把主页作品卡片当作目标账号作品。
5. 不把推荐视频、热门内容、大家都在搜、猜你喜欢等区域当作目标账号作品。
6. 如果作品未全部展开，记录已读取数量和缺口，不要无限滚动或高频刷新。

## 视频详情页读取流程

1. 打开用户提供的公开视频链接。
2. 完成非登录态检查。
3. 读取公开视频编号、标题正文、作者信息、发布时间、公开互动数值和话题标签。
4. 如果页面自然展示少量公开评论样本，可以记录样本并标注“公开可见评论样本，不代表完整评论区”。
5. 如果出现“请先登录后发表评论”“登录后即可参与互动讨论”等提示，记录为未登录信号，不要登录。
6. 如果出现登录弹窗或播放限制，只记录公开可见字段和限制原因。
7. 不下载视频，不读取隐藏媒体地址，不提取音视频资源。
8. 涉及评论时，只能按“公开评论样本读取流程”有限点击“展开回复”和“点击加载更多”；不得点击“立即登录”，不得为了评论继续深挖页面。

## 公开评论样本读取流程

1. 优先读取未登录公开页已经自然展示出来的评论文本。
2. 可以记录评论内容、公开可见时间地区、公开可见点赞数、作者公开回复。
3. 可以把评论样本归纳为用户关注点，例如安装门槛、成本、模板、教程、交付质量、变现可信度。
4. 必须写明样本数量和来源限制。
5. 在确认页面仍为未登录公开视角后，可以低频点击公开可见的“展开回复”，用于读取少量二级回复样本。
6. 在确认页面仍为未登录公开视角后，每条视频最多点击 3 次“点击加载更多”；如果按钮不可点击、没有新增内容或页面状态不明，停止尝试并记录限制。
7. 每次点击“展开回复”或“点击加载更多”后，都要重新检查是否出现登录弹窗、验证码、扫码、手机号输入或页面阻断；一旦出现，立即停止该动作，不得继续。
8. 不得称其为完整热门评论，不得称其为完整粉丝画像。
9. 不得登录后查看、批量翻页、高频刷新、无限滚动或为了评论追求全量。

## 搜索读取流程

1. 记录搜索关键词。
2. 优先使用非登录公开搜索页或公开搜索引擎结果。
3. 每个候选账号都必须保留可复核线索。
4. 搜索摘要中的粉丝、获赞、作品数只能作为待复核信息。
5. 输出时必须标注“搜索结果仅为候选池，不代表最终核验”。
