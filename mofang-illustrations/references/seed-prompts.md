# Seed Briefs

这些 brief 只提供起点。每次都要根据当前内容重新发明物件、动作和空间关系。

## 看见信号，不追噪音

- 核心意思：长期优势来自识别少数真实信号。
- 墨方动作：校准低科技过滤装置，把噪音线团压缩成少数稳定信号。
- 禁止：复刻望远镜与人群的左右布局。

## 小输入撬动大输出

- 核心意思：杠杆来自正确的系统支点，而不是蛮力。
- 墨方动作：拧动一个小旋钮，让系统机器持续输出整齐模块。
- 禁止：复刻跷跷板或巨大魔方。

## 多个 agent，一个 vision

- 核心意思：多个 agent 通过共同目标、规则和反馈形成合力。
- 墨方动作：在低科技调音台上协调不同 agent 声道，将其汇入一个清晰输出。
- 禁止：圆环、六宫格或组织架构图。

## 写作让思考变成系统

- 核心意思：写作把模糊想法压缩成可复用结构。
- 墨方动作：整理散乱纸带，将它们穿过桌面装置，输出少量系统卡片。
- 禁止：复刻云团到 framework/rules/process/system 的现有构图。

## 学习、迭代、复利

- 核心意思：学习通过反馈和修正积累成长期能力。
- 墨方动作：维护一台会把旧错误回流为新模块的学习机器。
- 禁止：复刻 chaos → pattern → structure → order 阶段图。

## 社媒头像

- 目标：圆形裁切后仍可识别。
- 保留：圆头、圆框眼镜、半眯眼、黑领结、触手、系统魔方。
- 禁止：世界观标签、复杂能力图标、长文案和商业光泽。

## 世界观设定图

- 目标：用动作和图标表达战略、世界模型、编排、杠杆、复利、学习和资源整合。
- 保留：墨方居中并真正协调模块。
- 禁止：照搬旧设定图的英文文案与布局、过满的 PPT 网格。
